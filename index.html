À propos
Suite d'Applications POO

Cette suite d'applications a été développée pour démontrer les concepts de la Programmation Orientée Objet (POO) en JavaScript.

Chaque application est encapsulée dans sa propre classe, suivant les principes de la POO :

  • Encapsulation : Chaque application gère son propre état et ses propres méthodes
  • Héritage : Organisation du code en classes réutilisables
  • Polymorphisme : Interface commune pour toutes les applications (init, destroy)
